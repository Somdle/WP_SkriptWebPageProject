Ban

사용예 ban %texts/offline players% [(by reason of|because [of]|on account of|due to) %text%]

unban %texts/offline players%

ban %players% by IP [(by reason of|because [of]|on account of|due to) %text%]

unban %players% by IP

IP(-| )ban %players% [(by reason of|because [of]|on account of|due to) %text%]

(IP(-| )unban|un[-]IP[-]ban) %players%

버전 1.4, 2.11 (밴 사유 사용시)
플레이어를 밴 할때 사용

예시

unban player

ban "127.0.0.1"

IP-ban the player because "he is an idiot"

Broadcast

사용예 broadcast %texts% [(to|in) %worlds%]
버전 1.0
서버내의 모든 유저를 대상으로 메세지를 전송함

예시

broadcast "hello every one"

Cancel Event

사용예 cancel [the] event

uncancel [the] event

버전 1.0
실행되는 이벤트를 취소함

예시

on damage:

  if player haver permission "admin":

    cancel event

Command

사용예 [execute] [the] command %texts% [by %players/console%]

[execute] [the] %players/console% command %texts%

(let|make) %players/console% execute [[the] command] %texts%

버전 1.0
외부 명령어를 실행

예시

on damage:

  if player haver permission "admin":

    cancel event

Delay

사용예 (wait|halt) [for] %time span%
버전 1.4
실행을 늦춤

서버의 모든 움직임과 행위는 틱(tick) 단위로 실행되며 1틱은 1/20초 입니다.

예시

wait 2 minutes

halt for 5 minecraft hours

wait a tick

Drop

사용예 drop %item types/experience point% [%directions% %locations%]
버전 1.4
아이템을 드롭함

블럭을 부쉇을 때 이벤트를 캔슬하고 drop 을 사용함으로 써 드롭하는 아이템을 바꾸는 형식으로 사용할 수 있음

예시

on death of creeper:

  drop 1 TNT

Exit

사용예 (exit|stop) [trigger]

(section|loop|conditional)

(section|loop|conditional)s

(section|loop|conditional)s
버전 알 수 없음
이벤트를 중간에 멈출 때 사용

예시

~~~~ if ~~~~:

  exit

Kick

사용예 kick %players% [(by reason of|because [of]|on account of|due to) %text%]
버전 1.0
플레이어를 추방할 때 사용

예시

on place of TNT, lava, or obsidian:

  kick the player due to "You may not place %block%!"

  cancel the event

Kill

사용예 kill %entities%
버전 1.0
엔티티를 죽일 때 사용

예시

kill the player

kill all creepers in the player's world

kill all endermen, witches and bats

Log

사용예 log %texts% [(to|in) [file[s]] %texts%]
버전 2.0
서버의 로그를 기록할 때 사용

외부폴더에 로그를 기록 할 수 있음

예시

log "%player% placed TNT in %world% at %location of block%" to "tnt/placement.log"

Message

사용예 (message|send [message]) %texts% [to %players/console%]
버전 1.0
플레이어 개인에게 메세지를 보낼 때 사용

예시

message "A wild %player% appeared!"

Op/Deop

사용예 [de[-]]op %offline players%
버전 1.0
플레이어에게 오피권한을 주거나 박탈할 때 사용

예시

the player

deop all players

Push

사용예 (push|thrust) %entities% %direction% [(at|with) (speed|velocity|force) %number%]
버전 1.4.6
엔티티를 특정 방향으로 밀어낼 때 사용함

예시

push the victim downwards at speed 0.5

PvP

사용예 enable PvP [in %worlds%]

disable PVP [in %worlds%]
버전 1.3.4
지역의 PvP여부를 지정함

예시

enable PvP #(해당월드에만 적용)

disable PvP in all worlds

Shoot

사용예 shoot %entity types% [from %living entities/locations%] [(at|with) (speed|velocity) %number%] [%direction%]

(make|let) %living entities/locations% shoot %entity types% [(at|with) (speed|velocity) %number%] [%direction%]
버전 1.4
오브젝트를 발사할 때 사용함

예시

shoot an arrow

make the player shoot a creeper at speed 10

shoot a pig from the creeper
