일반적인 루프는 아래와 같다

1.  loop 5 times:

2.    if loop-value is "ABC" or "DEF":

3.       message "루프값 : %loop-value%"

루프는 여러 값에 대해 코드를 반복하는 구문이다.

루프는 주어진 표현식의 모든 요소를 반복할 수 있다.(Ex: All players, worlds, items 등) 루프 내부의 조건과 효과는 모든 요소에 대해 실행되며 'loop-내용'을 통해 액세스 할 수 있다. 루프 내부의 조건이 충족되지 않으면 루프는 다음 루프요소를 시작한다. 그러나 stop loop 를 사용해 루프를 완전히 종료하고 코드실행을 계속 할 수 있다.

간단한 루프는 루프 표현식을 포함해 단일 조건이나 효과로 대체할 수 있다. (Ex. 모든플레이어를 루프해 아이템을 제공 을 모든플레이어에게 아이템을 제공)

1.   loop all players:

2.     give 64 of dirt to loop-player

2.     message "%player%님이 아이템을 배포" to loop-player

이 구문을 아래처럼 간단히 만들 수 있다.

1.   give 64 dirt to all players

루프가 가능한 값

둘 이상의 값을 나타내는 모든 표현식 (Ex. 목록변수, All players, worlds 등)

리스트 변수

리스트 변수를 루프시킬 때 루프하는 value 대신에 루프 index를 사용할 수 있다. loop-value 는 현재 루프된 변수의 값이며 loop-index는 변수 이름의 마지막 부분 (Ex. {Temp::*} 에서 *부분) 아래의 루프는 value 값을 1 씩 증가시키는 예제이다.

1.  loop {var::*}:

2.    set {var::%loop-index%} to loop-value + 1

While 루프

2.0 버전부터 while루프, 즉 조건이 충족될 경우 계속 반복하는 루프를 사용할 수 있으며 다음과 같다.

1.   while condition:

2.     code

while루프를 사용할 때에는 매우 조심히 사용해야 한다. while루프는 특정 조건을 만족하면 계속 실행되기 때문에 잘 못 다룰 경우 서버성능에 치명적인 영향을 줄 수 있다.
