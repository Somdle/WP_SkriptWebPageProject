Skript의 변수 소개

변수는 Skript의 데이터 저장 방법이며 모든 변수에는 이름과 값, 즉 일부 저장된 데이터가 있다.

트리거의 어느곳에나 변수를 넣어 무언가를 대체할 수 있다.

변수에는 영어, 한글, 숫자, 특수문자 모두 사용할 수 있다.

Ex) /home 명령어에 플레이어의 위츠를 저장시킬 수 있다.

변수와 일반 트리거를 구분하기 위해서는 {_temp}처럼 중괄호로 묶으면 되며 변수에 표현식이 포함될 수 있다.

Ex) {playerHome::%player%}, {%player%}, {playerUUID.%player's uuid%}

모든 변수는 플러그인 폴더의 "variables.csv"파일에 저장된다.

2.0전부터는 MySQL과 같은 데이터베이스에 연결 할 수 있다.

변수의 종류

변수는 일반변수, 지역변수, 옵션변수 3가지가 있으며 일반변수에 리스트 변수가 포함되어 있다.

일반변수는 서버가 종료되어도 사라지지 않고 스크립트 폴더에 저장된다.

지역변수는 이벤트나 명령어가 종료될 경우 사라지는 변수다.

옵션변수는 스크립트파일에서 지정하는 변수이며 해당파일에서만 적용된다. 이러한 옵션변수는 @를 사용해 어디서든 접근이 가능하며 %를 사용하지 않는다. Ex) message "{@main}" to ops

일반변수/리스트변수

일반변수

일반변수는 "variables.csv"에 저장되기 때문에 서버가 종료되어도 사라지지 않고 유지된다.

일반변수는 단순히 중괄호 { } 를 사용해서 만들 수 있다. Ex) {변수}

리스트변수

리스트변수 또한 "variables.csv"에 저장되며 서버가 종료되어도 사라지지 않고 유지된다.

리스트 변수는 중괄호 { } 와 쌍점 두개를 사용해 만들 수 있다. Ex) {변수::*}

{변수::*}에서 ::는 변수이름과 변수index를 구분하기 위해서 사용한다.

변수를 추가하려면 add "A" to {변수::*} 과 같이 사용이 가능하며 연속적으로 사용하면 추가적으로 계속해서 늘어난다.

변수를 읽어로려면 {변수::1}, {변수::2}와 같이 사용하면 읽을 수 있으며 loop를 사용해 편하게 사용할 수 있다.

{변수::*}에서 *은 반드시 숫자일필요는 없으며 player, world 등 여러가지가 들어갈 수 있다. Ex) {변수::%player%}

변수의 크기를 읽어오려면 size of 를 사용해서 읽어올 수 있다. Ex) size of {변수::*}

지역변수

지역변수는 서버가 종료되거나 이벤트, 명령어가 종료될 경우 사라지게된다.

지역변수는 중괄호 { } 와 언더바 _ 를 사용해 만들 수 있다. Ex) {_변수}

옵션변수

옵션변수는 스크립트 파일에서 지정할 수 있으며 해당파일에서만 적용된다. 옵션변수에는 %를 사용하지 않는다.

옵션변수는 다음과같이 선언할 수 있다.

option:

  일: 1

  이: 2

옵션변수는 다음과같이 사용할 수 있다.

message "{@일} 하나"
