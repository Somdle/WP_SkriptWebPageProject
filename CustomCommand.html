명령어의 구성

1. command 명령어 <내용 1> [<내용 2>]:

2.   usage: 명령어의 사용법

3.   permission: 명령어 펄미션

4.   permisison message: 펄미션 권한이 없을경우 나오는 메세지

5.   executable by: player/console

6.   aliases: /command1

7.   trigger:

8.     실행될 코드

1번에서 command는 명령어를 만들 때 사용한다. 명령어는 서버에서 사용될 명령어를 적는 곳이다. <와>로 묶인곳은 필수로 적어야할 공간에 사용한다. [<와>]로 묶인곳은 선택적으로 적어야할 공간에 사용한다. <와>, [<와>]를 사용해서 추가적으로 더 늘릴 수 있다.

6번은 추가적으로 사용할 명령어를 적는 곳이며 보통 단축명령어를 적는다.

1번과 7번은 필수이며 1번~6번은 필수가 아니다. 또한 8번에 실행될 코드를 적어넣는다.

Place Holder

위의 명령어의 구성에서 내용 1과 내용 2에는 명령어를 받을 타입을 넣는곳이다.

명령어를 받을 타입에는 item, player등이 있다.

사용예

command banplayer <player> [<text>]:

command /give <items> to <player>:

Argument

명령어에서 적은 내용을 불러오려면 arg 를 사용하면 된다.

사용예

if arg-1 is stick or dirt:

if arg-5 is "전송":

if arg-1 is not arg-2:
