Biome

사용예 biome[s]
버전 1.4.4
월드의 바이옴을 알아낼 때 사용(바이옴 = 지형)

예시

biome at the player is desert

player's biome is desert

Block

사용예 block[s]
버전 1.0
월드의 블럭 위치와 유형이 있으며 방향, 인벤토리등의 특수속성을 가지기도 한다.

예시

event-block

Date

사용예 date[s]
버전 1.4
게임내부의 날이아닌 현실의 날짜를 기준으로 한다.

예시

set {_temp} to now

subtract a day form {_temp}

Game Mode

사용예 game[ ]mode[s]
버전 1.0
게임모드에는 survival, creative, adventure 가 있다.

예시

player's gamemode is survival

set the player argument's game mode to creative

Item/Material

사용예 item

material
버전 1.0
아이템

예시

set {_item} to type of the targeted block

{_item} is a torch

Location

사용예 location[s]
버전 1.0
월드 내부의 위치 location 에는 월드, 좌표, 보는방향 이 저장된다.

예시

event-location

location of player

Text

사용예 (text|string)[s]
버전 1.0
텍스트로 사용하려면 큰따옴표("")로 묶어주면 된다.

예시

broadcast "Hello World!"

message "The id of ""%type of tool%"" is %id of tool%."

broadcast "Hello!" to "world_nether"
